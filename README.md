# Implemented using
- .NET Core
- SqlServer (InMemory)
- IoC
- Repository pattern
- Unit of Work
- Api Versioning
- Global Exception Handling
- SOLID
- Swagger
- EntityFramework Core
- Click counter from each url
- Save information such IP, Host and OS from each visitor
- Azure Pipelines CI
- Docker
